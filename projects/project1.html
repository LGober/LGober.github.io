Huffman Tree Implementation (Java)

Description: Built a Huffman tree based encoder for IoT-style message transmissions by analyzing symbol frequencies from input files, constructing a priority-queue Huffman tree, and generating prefix-free binary codes to reduce average transmission size and compare results when compared with fixed-length encoding.

Skills Learned

  • Priority queues / heaps
  • Tree traversal and recursion
  • Hash maps and frequency tables
  • Algorithm analysis / tradeoffs
